隐私协议

生效日期： [日期]

感谢您使用我们的应用（以下简称“本应用”）。 我们重视您的隐私，并承诺保护您的个人信息。请仔细阅读本隐私协议，以便了解我们如何收集、使用、保护您的信息。

1. 信息收集与使用

本应用完全依赖本地离线操作，不涉及任何联网功能。因此，我们不会主动收集、存储或共享您的个人信息、位置信息或任何其他数据。本应用仅在本地设备上运行，所有操作、数据传输和地图位置信息都仅在您的设备内部完成。

2. 数据存储与传输

本应用不进行任何形式的网络连接或数据上传，所有数据存储和处理都发生在您的设备本地。我们不会向任何第三方发送、共享或出售任何用户数据。

3. 不收集的个人信息

我们不会收集您的姓名、电子邮件、电话号码或任何身份识别信息。

我们不会收集您的位置信息，除非您主动使用设备的定位功能进行调试操作。即使使用了定位功能，所有位置信息也仅限于设备本地使用，不会上传至任何外部服务器。

4. 使用权限

本应用需要访问蓝牙功能以连接外设进行调试。

本应用需要访问设备的定位功能，以传输地图位置信息。所有定位信息的处理完全在本地进行，未通过网络传输。

5. 数据安全

由于本应用不进行数据上传或联网功能，所有信息都保存在您的设备本地。我们建议您妥善保管您的设备密码、蓝牙连接权限等安全设置，以保护您的隐私安全。

6. 未成年人保护

本应用并未专门面向未成年人。我们不收集未成年人的个人信息。如您是未成年人的监护人，请确保未成年人在使用本应用时获得您的同意。

7. 变更与更新

我们可能会随时更新本隐私协议，以反映新的功能或更改。我们会通过本应用内更新或公告通知您任何重大更改。请定期查看本协议，以了解我们的隐私政策是否发生变化。
